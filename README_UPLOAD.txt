1. Распакуй архив.
2. В папке src открой App.jsx и ПОЛНОСТЬЮ замени его содержимое кодом из твоего скачанного файла offermatch_samiy_norm_duplicate_preview (5).jsx.
3. В этом вставленном коде замени:
   process.env.NODE_ENV !== "production"
   на:
   import.meta.env.DEV
4. Залей все файлы из архива в корень репозитория OM с заменой существующих.
5. В Settings -> Pages выбери Source = GitHub Actions.
6. Открой Actions и дождись зеленого Deploy to GitHub Pages.
7. Сайт откроется по https://johnnydeppisnumber1.github.io/OM/
